# Excel PandaPolar

El análisis de datos con archivos excel.

## Descripción

Este repositorio contiene scripts en Python para el análisis de datos utilizando archivos Excel. Utilizamos la librería `pandas` para manipular y analizar los datos de manera eficiente.

## Estructura del Proyecto

- `Analisi_Pyton_OrdenCompra2024`: Contiene scripts específicos para el análisis de órdenes de compra del año 2024.

## Requisitos

- Python 3.x
- pandas

## Instalación

1. Clona este repositorio:
    ```bash
    git clone https://github.com/Zuricato01/Excel_PandaPolar.git
    ```
2. Navega al directorio del proyecto:
    ```bash
    cd Excel_PandaPolar
    ```
3. Instala las dependencias:
    ```bash
    pip install pandas
    ```

## Uso

Ejecuta los scripts de análisis como se indica en la documentación de cada archivo.
